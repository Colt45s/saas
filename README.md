# saas

## Backend

### setup

```
$ cargo install cargo-watch
```

### Docker

```
$ docker-compose up -d
```

### Prisma

```
$ cargo prisma db push
$ cargo prisma generate
```

### run

```
$ cargo run
```

### watch

```
$ cargo watch -x run
```

## Frontend

### setup

```
$ yarn
```

### run

```
$ yarn dev
```
